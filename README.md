# Faker 
___
> Faker Library for PHP 7.4/8.2

## Install
```
$ composer require mubiridziri/faker
```

## Example

```php

$faker = new \Mubiridziri\Faker\Service\FakerManager();
$result = $faker->{useAnyPublicMethods}()

```
